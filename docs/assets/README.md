## Assets

You can serve static assets from the CDN, or locally.

### Variables

| Env var                | Description            | Supported values | Default value |
| ---------------------- |:----------------------:|:----------------:|:-------------:|
| ```WUD_LOCAL_ASSETS``` |  Serve assets locally  | 1 0              | 0             |

### Examples

#### Serve assets locally
```bash
WUD_LOCAL_ASSETS=1
```
